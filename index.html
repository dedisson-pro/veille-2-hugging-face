1.Presentation

Hugging Face est une plateforme d’intelligence artificielle et de machine learning qui permet à la communauté de partager, découvrir et utiliser des modèles IA pré‑entraînés, des jeux de données et des applications. Elle est souvent comparée à un “GitHub pour les modèles IA” car elle favorise la collaboration et la réutilisation de ressources open source.

2. Quelles sont les fonctionnalités de Hugging Face ?

  • Model Hub – un grand dépôt de milliers de modèles IA.
  • Datasets – une bibliothèque de jeux de données
  • Spaces – interface pour créer des applications IA interactives.
  • Transformers – bibliothèque Python pour IA et NLP

3. Qu’est‑ce que Hugging Face permet réellement de faire ?

  • 🔎 Utiliser des modèles IA déjà entraînés (texte, images, audio)
  • 🔁 Fine‑tuner (adapter) ces modèles à ton jeu de données.
  • 🚀 Déployer des applications d’IA interactives via Spaces.
  • 🧠 Explorer et partager jeux de données et modèles dans la communauté.

4. Comment créer son premier projet avec Hugging Face ?

  1. 👉 Crée un compte sur huggingface.co.
  2. 👉 Explore les modèles disponibles dans le “Model Hub”.
  3. 👉 Installe la bibliothèque Transformer en Python : 
    pip install transformers datasets
  4. 👉 Utilise un modèle dans ton code : 
    from transformers import pipeline
classifier = pipeline("sentiment-analysis")
print(classifier("J’aime Hugging Face !"))
  5. 👉 (Optionnel) Crée un Space pour ton projet avec Gradio ou Streamlit.
